# Einfach

## So funktioniert wiCow für Sie.

Nein, wirklich. Die Details sind recht komplex, aber wir möchten Ihnen diese 
Details entziehen, damit Sie sich auf andere Bereiche konzentrieren können. 
Alles, was Sie bei der Verwendung des wiCow-Systems beachten müssen, sind 
drei einfache Schritte.

1. Installieren Sie Sensoren an den Kühen. Wicow wird Sie zum richtigen Zeitpunkt 
daran erinnern, oder Sie können es jederzeit tun, wenn Sie dies bevorzugen. 
*Die Installation eines TSens-Sensors bei einer Kuh dauert ungefähr fünf Minuten*.

2. Machen Sie andere Dinge auf Ihrem Bauernhof oder im Freien, während wiCow 
Ihre Kühe 7/24 mit einer ausgeklügelten künstlichen Intelligenz verfolgt. 
*Wicow erkennt das Kalben 24 Stunden bevor es passiert*, Sie haben also 
genügend Vorlaufzeit, um dies zu planen.

3. Ergreifen Sie Maßnahmen, wenn Wicow Ihnen eine Benachrichtigung sendet. 
Bei Fieber wissen Sie genau, wie es passiert, und beim Kalben erhalten Sie 
zwei Warnungen, eine innerhalb von 24 Stunden, eine weitere zu Beginn. 
Dann *machen Sie einfach so, wie Sie es früher getan haben*.
